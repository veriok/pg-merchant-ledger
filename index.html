📋 Changelog

v1.0.1

  • Cash sort now places vendors with no cash tracked at the end instead of the top
  • Region sort now follows fixed zone order (Serbule → Eltibule → Serbule Hills → Casino → Rahu → Kur → Sun Vale → Ilmari → New Prestonbule Cave → Povus → Statehelm → Fae Realm)
  • Timers: expired timer with a cash amount set now automatically resets cash and clears the timer
  • Per-vendor limits override: click 'Modify Limits' (in the Manage tab, per vendor) to override item cap / cash pool per favor level; saves to local storage, falls back to wiki defaults
  • Item Sell Cap (%) field in toolbar — multiply all displayed item caps by a percentage (e.g. 117% for skill/effect bonuses); saves to local storage

v1.0.0

  • Initial release
  • Vendor selling panel with favor tracking, cash remaining, and weekly pool
  • Gifting panel with per-vendor gift tables and favor-to-next calculations
  • Manage panel: customize vendor buy types and multipliers
  • Timer system: track time until vendor resets
  • Favorites, beast speech filter, sold-out filter, item type filter
  • Export / import favor data as JSON
  • All data saved locally in the browser (localStorage)
